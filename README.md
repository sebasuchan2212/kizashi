# KIZASHI｜確率叙事占学

KIZASHI is a React + TypeScript + Vite MVP for a self-reflection reading app based on 確率叙事設計学.

## Updated design

This version rebuilds the homepage to match a luxury cosmic landing page style:

- Dark navy cosmic background
- Gold and cyan accents
- Sticky navigation
- Hero section with orbital astrolabe visual
- Feature cards
- 3-domain cards: 恋愛 / 仕事 / ウェルネス
- Sample diagnosis preview
- Functional diagnosis form
- Scenario result cards
- Pricing plan selection UI
- FAQ
- Safety-support link

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel settings

- Framework Preset: Vite
- Install Command: npm install
- Build Command: npm run build
- Output Directory: dist
- Node.js Version: 20.x

## Notes

Do not commit `node_modules`, `dist`, `package-lock.json`, or `tsconfig.tsbuildinfo` unless intentionally needed.
